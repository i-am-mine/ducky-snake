## What it is

This is a duck-themed version of the classic Snake game. Control a snake, represented by a Mother duck (ğŸ¦†) for the head and children ducks (ğŸ¥) for the body, as it moves around a 10x20 grid, collecting vegetables to grow longer and increase your score.

## How to Play

- Use **W-A-S-D** keys to control the snake's direction
- Press **X** to quit the game
- Eat a vegetable to get 10 points and an egg. Over time, the eggs will hatch into ducklings.
- Avoid hitting the walls or your own body
- Ducks chain cannot go backward.

## How to Run
Run the file:
```bash
./run.sh
```

## Game Controls

- **W** - Up
- **A** - Left
- **S** - Down
- **D** - Right
- **X** - Quit game
